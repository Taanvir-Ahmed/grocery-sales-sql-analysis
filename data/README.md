## Dataset Information

The raw CSV files are not included in this repository due to size limitations.

Dataset source:
Grocery Sales Database (Kaggle)
URL: https://www.kaggle.com/datasets/andrexibiza/grocery-sales-dataset

Files required:
- categories.csv
- cities.csv
- countries.csv
- customers.csv
- employees.csv
- products.csv
- sales.csv (large file)

Download the dataset from Kaggle and place the CSV files in this `data/` directory
before running the SQL scripts.
